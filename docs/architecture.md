# S.T.A.R.K. Architecture

## Hierarchy
- **Father:** STARK (director, coordinator)
- **4 Daughters:** Morgan (coding), Peter (research), Harley (small tasks), Ultron (defense)
- **12 Kids:** 3 specialized agents under each daughter

## Communication Flow
You → DeepSeek (translator) → Father STARK → Daughter → Kids → Daughter → Father → You

## Zero-Budget Stack
- Python 3.10+
- Vosk (offline voice)
- DeepSeek API (free translator)
- Flask (dashboard)
- SQLite (memory)
